### _build

clone the repository,

```bash
$ git clone https://github.com/yogurt-foundation/yogurt-css-documentation.git
$ cd yogurt-css-documentation
```

install dependencies,

```bash
# YARN
$ yarn
# NPM
$ npm i
```

serve the dev project,

```bash
# YARN
$ yarn dev
# NMP
$ npm run dev
```
