## Usage

Set font-size to an element.

```html
<!-- Example -->
<y class="text-md">
  ...
</y>

<span class="text-md">
  ...
</span>
```

Set default font-size with `<body>` tag.

```html
<!-- Example -->
<body class="text-md">
  ...
</body>
```
