# Font Weight

Utilities for sets the font weight.

---

## Usage

Set font-weight to an element.

```html
<!-- Example -->
<y class="font-light">
  ...
</y>
```

Set font-weight with `Google Fonts`. (See [Font Family](/font-family/#customize-more)).

```html
<!-- Example -->
<y class="font-roboto font-light">
  ...
</y>
```
